<!-- Run Locally -->
### :running: Run Locally

Go to the project directory

```bash
  cd my-project
```

Install dependencies for client

```bash
  cd seafood-site
  npm install
```
  
  
Install dependencies for server

```bash
  cd backend/seafood-site
  npm install
```

Start the server

```bash
  cd seafood-site/backend
  npm start
```
